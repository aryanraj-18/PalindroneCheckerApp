# PalindromeCheckerApp

This is a Java application to check if a given string is a palindrome.

## Features

- Welcome message on startup
- Checks hardcoded palindromes
- Case-insensitive and ignores spaces

## How to Run

1. Compile the Java file: `javac App/src/PalindromeCheckerApp.java`
2. Run the application: `java -cp App/src PalindromeCheckerApp`

## Use Cases

- UC1: Application Entry & Welcome Message
- UC2: HardCodePalindrome
